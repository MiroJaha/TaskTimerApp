# TaskTimerAppilcation
Android Application to Manage People Time
  
   
**Demo:**
[![Task Timer Application](https://user-images.githubusercontent.com/90371913/152640533-91cdf6d0-42bf-4ceb-aa57-b707feab631a.jpg)](https://youtu.be/FP46W2l5UUU)  

Using **Kotlin**  

This Code Provide Simple **Android Application** to **Manage People Time**

This Application Called **Task Timer Appilcation**

This Application Uses **SQLite**

**This Application Will:**
* • Use SQLite as Database
* •	Read From SQLite
* •	Write to SQLite
* •	Delete From SQLite
* •	Save in SQLite
* • Calculate Total Task Time
* • Use Pie Chart for All Tasks
* • Uses Swiping to Swich Between Views
&nbsp;  
&nbsp;  

**This Application Contains 5 Views:** 

**1. Instruction View:**
*     •	Start Instruction Journey
*     •	Instructions for Using the Application
*     •	Explain Evey View
*     •	Option to Skip Instructions Next Time the Application Started
